Quasar Selectbox     

A Quasar Selectbox can be of two types: single selection (using Radios) or multiple selection (using Checkboxes).

Basic Usage

Vue DOM attribute properties you can use:

Param AttributesDescription
modelModel for the Web Component.
optionsArray of options (Object with label and value properties).
multipleEmpty attribute. When present it allows for multiple selection.
ok-labelLabel for “OK” button
cancel-labelLabel for “Cancel” button
titleTitle displayed when choosing option(s).
messageMessage below the title when choosing option(s).
fixed-labelLabel to display on the Web Component to inform user of the current selection. The default label can be replaced with a String
