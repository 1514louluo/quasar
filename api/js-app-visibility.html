Quasar App Visibility     

Quasar makes use of the Page Visibility API which lets you know when a website/app is visible or in focus.

There is also an event triggered when focus changes:

Event NameParametersDescription
app:visibility(String) stateApp got focus if state is visible or lost focus if state is hidden
