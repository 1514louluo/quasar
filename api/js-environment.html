Quasar Environment     

Helpers are built-in to detect the environment in which the code is running:

PropertyTypeMeaning
quasar.runs.on.mobilebooleanIs the code running on a mobile device?
quasar.runs.on.cordovabooleanIs the code running within Cordova?
quasar.runs.on.desktopbooleanIs the code running on a desktop browser?
quasar.runs.with.touchbooleanIs the code running on a touch capable screen?
quasar.runs.within.iframebooleanIs the App running within an IFRAME?

NOTE
Running on mobile means you can have this code running on a mobile device (phone or tablet) but with a browser, not within a Cordova wrapper.
