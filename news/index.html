Quasar Framework

Modern HTML5 SPA front-end framework on steroids. With VueJS.

Desktop and Mobile. Native Mobile Apps included. Full Stack Toolkit.

All Platforms in One Go

Learn the Quasar way and the framework takes care of the quirks for each platform. You'll have just one source code for all platforms simultaneously: desktop/mobile website, native mobile apps (iOS, Android, and so on).

Speed and Efficiency

Most of the heavy-lifting is already done through Quasar CLI. Your focus is on your App content rather how the App is built and optimized for each platform. Quasar takes care of the redundant tasks allowing you to be the most efficient possible.

Web Components

Quasar Web Components are at the heart of the framework. Contribute or use one from the built-in ecosystem. Write as few lines of code as possible and yet enrich your App with beautifully crafted functionality with a unified user experience across mobile/desktop platforms.

Time Saver

Out of the box code boilerplating, build system, live preview, test setup, image optimizations and many more. You don’t have to worry about a lot of things, including how your content is being displayed to best leverages the screen real-estate. The built-in components are responsive by default.

Two-Way Data Binding

Write smart reactive apps. No need to directly touch DOM. Write custom HTML tags or use the built-in ones. Javascript files use CommonJS format to ensure encapsulation and a clear separation of concerns.

Build System (& CLI)

Development with sourcemaps or minified & optimized for production out of the box, no configuration needed. Bundles the source code in such a way that it ensures the smallest footprint possible and best performance.

Wide Support

iOS, Android, Windows Phone, Blackberry, Google Chrome, Firefox, IE/Edge.

Themes

Quasar uses its own full-featured CSS Framework. You can tweak it and build your own theme to make your App’s design distinctive. The built-in themes are Android and iOS which can be extended within each App. Don't worry about auto-prefixing browser specific CSS properties as this is taken care of by default.

Built for Performance

Quasar has a lot of optimizations behind the scenes to make your App run really fast (and consume the lowest possible battery power). It was carefully crafted with latest Web performance recommendations in mind.

Ensures Code Quality

Quasar CLI ensures code quality out of the box (linters, app validators, a ready to use automation testing setup).

Live Preview

Watch multiple browsers on multiple machines auto-refresh on source code change. Complex tool included in Quasar CLI to speed up your development. Responsive Live Preview is a also a feature built to help you quickly see how your App looks on different screen sizes.

Write Reusable Code

Use built-in reactive HTML tags as web components or write your own. It's easy to reuse them, even across multiple Apps.

Built-in Widgets

Use a large variety of built-in widgets/plugins or write your own and share them.

Powerful Layouts

Out of the box multiple layouts. Highly Customizable. Let Quasar worry about the layout for each environment (browser, Android, iOS and all others). You just focus on the content.

Complete Toolkit

Quasar offers API for Routers, Environment detection, Events, Ajax, Touch Events, Cookies, Web Storage, App Visibility, and many more! Everything you need in one place.

Great Documentation

Everything is documented in crystal clear fashion, with examples, guides, tips and demos with source code available.

Open Source

Visit Github page. Help with fixing bugs or write new features/widgets/plugins. Any suggestion is appreciated. Read how to contribute.

Dependencies

Quasar depends on Vue, Apache Cordova, HammerJS, VelocityJS, jQuery. Read more here.
