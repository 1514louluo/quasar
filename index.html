Quasar Framework

Modern HTML5 Single Page Application front-end framework on steroids.

Working on 1st release.

Build on one. Use it on all.

* Including native mobile Apps. *

Speeds Up Development

Most of the heavy-lifting is already done (through Quasar CLI). Your focus is on your App content rather how the App is built and optimized for each platform.

Focused on User Experience

Components provide great feedback to the user while the responsive user interface allows for a unified user experience across mobile/desktop platforms.

Great Documentation

Everything is documented in crystal clear fashion, with examples, guides and tips.

Time Saver

Out of the box code boilerplating, build system, live preview, test setup, image optimizations and many more.

Two-Way Data Binding

Write smart reactive apps. No need to directly touch DOM. Custom HTML tags.

Wide Support

iOS, Android, Windows Phone, Blackberry, Google Chrome, Firefox, IE/Edge.

Ensures Code Quality

Quasar CLI ensures code quality out of the box (linters, app validators, test setup).

Web Components

Contribute or use one from the built-in ecosystem. Write as few lines of code as possible and yet enrich your App with beautifully crafted functionality.

Live Preview

Watch click/scroll/form input synchronized browsers auto-refresh on source code change. Responsive View also available to quickly switch between desktop/tablet/phone screen sizes (with orientation too).

Write Reusable Code

Use built-in reactive HTML tags as web components or write your own. It's easy to reuse them, even across multiple Apps.

Build System

Development with sourcemaps or minified & optimized for production out of the box, no configuration needed. Bundles the source code in such a way that it ensures the smallest footprint possible.

Semantic API

Intuitive API syntax from natural languages.

Built-in Widgets

Use a large variety of built-in widgets/plugins or write your own and share them.

Powerful Layouts

Out of the box multiple layouts. Highly Customizable. Let Quasar worry about the layout for each environment (browser, Android, iOS and all others). You just focus on the content.

Open Source

Help with fixing bugs or write new features/widgets/plugins.
